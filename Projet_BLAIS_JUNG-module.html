draw_hierarchical_tree(tree, root, current_cluster)

source code 

Draw a hierarchical tree. Each clusters are represented by a subgraph. For each cluster add a node in the tree then fit the edges between the cluster and its parent recursively.

Then, for each current clusters that doesn't have childs draw each genes as nodes and fit it to its cluster.

Parameters:
  • tree (tlp.Graph) - graph holding root
  • current_cluster (tlp.Graph) - a Tulip graph of the hierarchical tree in recursivity
  • root (tlp.node) - root node of the current_cluster

compute_path(L1, L2)

source code 

Find the shortest path between two nodes using two lists. The lists are the path between the node of interest and the root node of the tree. The first common node between this two path is the closest node which links the source and the target.

This function allow to find the first common node between two lists and remove the other nodes. Then return the path between the source and the target.

Parameters:
  • L1 (list) - first path between the source node and root
  • L2 (list) - second path between the target node and root
Returns:
node list ordered as the shortest path
